HTML

  • HMTL is made up of elements that perform a distinct role. These are !DOCTYPE HTML, HTML, head, meta and title.
  • !DOCTYPE HTML is a declaration that tells te browser what document type it should expect when it is opened.
  • HTML is the opening tag that represents the start of the HTML document.
  • The head element contains information, or metadata, that helps the browser render the page correctly.
  • Elements within the head are not visible to the user.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users.
  • Metadata is typically not seen by users, but it contains important information about accessibility, search engines, and performance.
  • Title is literally the title of the webpage. This is important for SEO (search engine optimization)
  • The body element represents the visible content shown to the user.

CSS

  • Internal Style Sheet is a style element INSIDE the head element.
  • It contains CSS rules for the head section of the HTML file.
  • Most common way to style a website is external CSS style sheet.
  • A margin indicates how much space we want around the outside of the element.
  • A padding indicates how much space we want around the content inside an element.
